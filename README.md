## ✨ Features

**BeatLab Studio** is a high-performance web application that transforms your browser into a mini digital audio workstation (DAW). Built with **Tone.js** and **React**, it offers low-latency audio processing and a sleek, modern interface.

| Feature | Description |
|--------|-------------|
| 🥁 **Step Sequencer** | 16-step grid for Kick, Snare, Hi-hats, Bass, and Cowbell. |
| 🎹 **Poly Synth** | Interactive keyboard with dual oscillators and ADSR envelope. |
| 🎚️ **Mixer Console** | Individual track volumes, mute/solo controls, and master FX. |
| 💾 **Persistence** | Automatically saves your patterns and synth settings to local storage. |
| ⚡ **Tone.js Engine** | Professional-grade audio scheduling and synthesis. |
| 🎨 **Modern UI** | Crafted with Tailwind CSS for a responsive, dark-mode experience. |

## 🚀 Quick Start

### 1. Installation

```bash
# Clone the repository (or extract the zip)
cd BeatLab-Studio
npm install
```

### 2. Launch Development Server

```bash
npm run dev
```

### 3. Start Creating
Open `http://localhost:3000` and start clicking the sequencer steps to build your first beat!
